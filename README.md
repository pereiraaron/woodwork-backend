# Woodwork Backend

A backend service built with [NestJS](https://nestjs.com) and MongoDB.

## Setup

```bash
npm install
```

## Running

```bash
# development
npm run start

# watch mode
npm run start:dev

# production
npm run start:prod
```

## Tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
